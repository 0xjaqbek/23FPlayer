# Audio Relay

Standalone Node service for accepting one active browser broadcaster and forwarding audio chunks to authenticated listeners.

## Environment

```text
PORT=4010
RELAY_SHARED_SECRET=
RELAY_LISTENER_SECRET=
APP_HANDOVER_URL=
```

`RELAY_SHARED_SECRET` must match the web app for broadcaster tokens and internal relay calls. `RELAY_LISTENER_SECRET` must match the web app for short-lived listener stream tokens. Broadcasters connect to `/broadcast?token=<signed-broadcast-token>`; listeners use the signed stream URL generated by the web app.

## Commands

```bash
pnpm --filter relay dev
pnpm --filter relay test
pnpm --filter relay build
pnpm --filter relay start
```
